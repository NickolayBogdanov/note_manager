# 1. create_note_function.py: Функция создания заметки
#   Функциональность:
#     1) Запрашивает данные у пользователя для создания заметки.
#     2) Формирует словарь с полями заметки, включая автоматическую генерацию текущей даты.
#     3) Проверяет корректность формата даты дедлайна.
#   Пример работы программы:

#     Введите заголовок: Заголовок
#     Введите заметку: Заметка
#     Выберите статус заметки "1" - Выполнено, "2" - В процессе или "3" - Отложено: 2
#     Стаус успешно изменен, текущий статус:  В процессе
#     Введите дату истечения заметки в формате 'дд-мм-гггг': 30-01-2025
#     Заметка создана успешно!
#     Текущие заметки:
#     ----------------------------------------------------
#     Note №1
#     1) username        Имя                           
#     2) title           Заголовок                     
#     3) content         Заметка                       
#     4) status          В процессе                    
#     5) created_date    26-01-2025                    
#     6) issue_date      30-01-2025
